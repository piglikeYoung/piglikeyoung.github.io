前言

开发中我们经常需要检测某个方法或者某段代码具体的执行时间，有人使用NSDate来计算，但是这个类的精确度太差，你可以抛弃它了。

其实在iOS中有个mach_absolute_time()函数可以获取到CPU的tickcount的计数值（貌似Android也有个获取精确时间的函数），获取到纳秒级的精确度。
